# BLACK SIGNAL — 시즌 2 「TWIN TALLY」 설계 스펙

날짜: 2026-07-05 · 상태: 승인됨 (브레인스토밍 완료)
범위: 시즌 2 전체 아크(4챕터) 설계 + 첫 챕터(Chapter 7) 구현 상세

## 0. 원칙 (허구화 — 최우선 제약)

- 실제 대한민국 선거·실존 기관(중앙선관위, A-WEB 등)·실존 인물·정당과 **무관한 완전 허구**.
  무대는 가상 도시, 시스템·조직·인명은 전부 가상 명칭. 프로그램명도 실존 기관명(A-WEB)과
  겹치지 않도록 `ballast` 로 확정.
- "실제 투표 제도가 뚫렸다"가 아니라 **"가상 도시의 신규 시범 시스템이 오염됐다"** 구도 —
  종이 투표는 그대로, 집계·전송 단계만 디지털화한 파일럿이 무대.
- 시즌 1의 안전 원칙 유지: 라이브 시스템 접근 없음, 압수·제출된 **읽기 전용 사본**만 분석,
  매 챕터 caseSummary.safety 에 허구 고지.

## 1. 세계관·명칭 (전부 가상)

| 항목 | 명칭 | 비고 |
| --- | --- | --- |
| 무대 | **한서시** | 가상 광역시. 2026-10-14 한서시장 보궐선거, 14개 개표구 |
| 시범 시스템 | **TALLYBRIDGE** | 디지털 개표 집계 파일럿 (집계·전송만 담당) |
| 납품업체 | **Suncrest Systems** | 몸통이 아닌 경유지 — 인증 패치 단계에서 모듈 주입됨 |
| 부정 프로그램 | **ballast** (blst-0.9.4) | 소문자 코드네임 — 시즌 1 작명 문법(mirrorcall, dove-9) 계승 |
| 배후 컨설팅펌 | **Meridian Civic** | "결과 보증"을 상품으로 파는 가상 선거 컨설팅펌 |
| 후보 3인 | 기호1 차수완 · 기호2 백도현 · 기호3 임가려 | 전원 가상 인명 |
| 제보자 | **정다인** | 제11개표구 개표 사무원 |

**ballast 의 원리와 열쇠**: 배의 평형수처럼 득표율을 목표 분포로 '평형'시키는 은닉 모듈.
표를 만들지 않고 **옮긴다** (총투표수 보존). 정수 반올림 쿼터가 규모가 비슷한 개표구들에서
후보별 득표수를 자리수까지 똑같이 찍어냄 — **쌍둥이 개표수는 ballast 의 버그**이며,
완전범죄를 무너뜨리는 열쇠가 조작 프로그램 자신의 결함이라는 아이러니가 시즌의 척추다.

**시즌 1과의 연결(약한 연결)**: 주인공·NW·기자 서지원 등 인물 연속. Meridian 의 초기
투자금 관로에 Orbis Vale 잔당 셸(NL 계열) 흔적이 카메오로 등장 (E4).

## 2. 유저 제공 5비트의 배치

1. 투표지 부족 항의 → E1 도입 (제보자의 출발점)
2. 선관위(가상 한서시) 묵살·개표 강행 → E1 수사 대상 (항의 대장 대조)
3. 지역구별 쌍둥이 개표수 → E1 핵심 퍼즐 (통계 포렌식)
4. 부정선거 의혹 확산 → E1 엔딩~E2 배경 (부인 캠페인)
5. 정부(가상 발주 라인)의 파일럿 개표기 조작 → E2~E4 입증·배후 특정

## 3. 시즌 아크 (Chapter 7–10)

- **E1 · Ch7 「Twin Tally — 쌍둥이 표」**: 정다인 제보 → 묵살 정황 + 쌍둥이 쌍 통계 확정
  → 감사 로그에서 미승인 모듈 서명 `ballast` 첫 발견. 훅: 정다인 협박 전화.
- **E2 · Ch8 「Ballast」**: 압수 개표기 펌웨어 포렌식. 목표 득표율 테이블·반올림 쿼터로
  쌍둥이 표의 인과를 수학적으로 증명. 주입 경로 = Suncrest 내부가 아닌 '인증 패치' 서버.
- **E3 · Ch9 「인증의 사슬」**: 검수·인증 결재선 추적, 위조 검수 보고서, Meridian 접점.
  반전: 투표지 부족(비트 1)의 배부 수량 산정에 Meridian 의 투표율 과소 예측 모델이 쓰였음 —
  행정 실수처럼 보였던 첫 실이 같은 몸통으로 이어진다. 제보자 위협 고조(세리톤 계열 재등장).
- **E4 · Ch10 「조용한 압승」**: "표를 훔치지 않는다. 오차 범위 안에서 이기게 만들 뿐" —
  Meridian 의 '결과 보증' 상품 구조, 구매자(가상 정부 비선 발주 라인), 설계자 개인 특정.
  최종 선택: 재검표 청구 / 일제 공개 / 제도 이관 / 피해 지역구 주민 주체.
